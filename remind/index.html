Commands

Remind

Description Remind a user when they next type in chat, or set a timed reminder for a user (or yourself)
Command remind
Aliases notify
Cooldown 5 seconds
Admin only? âœ˜
Usage
Remind a user when they next type in chat:
> remind <user> <message>
> reddit forsen play assassin's creed black flag

Set a timed reminder to remind the user after a set duration in the current channel (can't be set in whispers):
> remind <user> in <time period> <message>
> remind forsen in 20 mins go live forsenSWA
> remind ClintStevens next month OOT speedrun?

Set a timed reminder for yourself:
> remind me in 10 minutes pasta
> remind me tomorrow update stream title
