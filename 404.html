alias > ✘ 5 
                
Runs a command using a custom alias.

Examples:
Add a new alias
> alias add <alias name> <commands>
> alias add forsenclip randomclip forsen

Remove an alias
> alias remove <alias name>
> alias delete <alias name>
> alias remove forsenclip

Update an alias
> alias edit <alias name> <commands>
> alias update <alias name> <commands>
> alias edit forsenclip randomclip forsen

Get an alias definition
> alias definition <alias name>
> alias definition forsenclip

Run an alias
> > <alias name>
> alias run <alias name>
