Usage

Add a new alias

> alias add <alias name> <commands>

> alias add forsenclip randomclip forsen

Remove an alias

> alias remove <alias name>

> alias delete <alias name>

> alias remove forsenclip

Update an alias

> alias edit <alias name> <command>

> alias update <alias name> <command>

> alias edit forsenclip randomclip forsen

Get an alias definition

> alias definition <alias name>

> alias check <alias name>

> alias spy <alias name>

> alias definition forsenclip

Copy an alias

> alias copy <username> <alias name>

> alias copy forsen randomForsenClip

Copy and overwrite an existing alias

> alias copyplace <username> <alias name>

> alias copyplace forsen randomForsenClip

Run an alias

> run <alias name>

> alias run <alias name>
