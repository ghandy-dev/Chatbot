Braille

Description Generate ASCII braille art of an image.
Command braille
Aliases
Cooldown 20 seconds
Admin only? âœ˜
Usage

Specify an image URL

> braille <url>

> braille https://static-cdn.jtvnw.net/emoticons/v2/25/default/dark/3.0

Specify an emote.

Can be a Twitch/BTTV/FFZ/7TV emote, but due to limitations; if a Twitch Emote is specified it must be one accessible to the bot (i.e. either it must be a global emote, or an emote from a channel the bot is subscribed to)

> braille <emote>

> braille Kappa

Specify a color intensity function:

Supported color intensity functions:

- lightness (default)

- luminance

- average

- max

> braille <function> <url>

> braille average https://static-cdn.jtvnw.net/emoticons/v2/25/default/dark/3.0

> braille <function> <emote>

> braille average Kappa
