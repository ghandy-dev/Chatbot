Commands

Random Clip

Description Get a random clip from the current channel, or a specified channel.
Command randomclip
Aliases rc
Cooldown 20 seconds
Admin only? âœ˜
Usage
Get a random clip from the current channel:
> randomclip

Get a random clip from a channel:
> randomclip <channel>
> randomclip forsen

Get a random clip using in a given period:

Valid periods are:
- day
- week (default)
- month
- year
- all

> randomclip <channel> period:<period>
> randomclip lirik period:year
